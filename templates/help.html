ACM班个人页面使用帮助

必要信息

ACM班个人页面旨在为ACM班学生提供一个可以放置个人学术网站的空间。

  • 管理页面： https://acm.sjtu.edu.cn/userpage-manage
  • 网站地址： https://acm.sjtu.edu.cn/~jaccount
  • 只支持静态页面
  • 索引顺序： index.html, index.htm, index.xml
  • 文件夹自动索引开启，如文件夹内没有上面文件之一，则会显示文件列表
  • 文件配额： 100MB
  • 文件上传方式： SFTP
  • SFTP 主机： acm.sjtu.edu.cn
  • SFTP 端口： 2222
  • SFTP 用户名： jAccount
  • SFTP 认证方式： SSH Key，可在管理页面进行管理
  • 推荐 SFTP 客户端： FileZilla
  • 推荐 SSH Key 生成程序： (Linux/Mac) ssh-keygen, (Windows) PuttyGen

Linux/Mac 生成 SSH Key

如果你之前没有生成过 SSH Key，可以运行 ssh-keygen。

Windows 生成 SSH Key

如果你之前没有生成过 SSH Key，可以下载 PuttyGen，点击 Generate。生成完毕之后，分别点击 Save public key 和 Save private key 保存公钥和私钥。

管理页面添加 SSH Key

打开管理页面，使用 jAccount 登入。在 SSH Key 管理页面中，填入你的 SSH Key 公钥。

  • Linux/Mac 获取公钥：cat ~/.ssh/id_rsa.pub 然后复制即可
  • Windows 获取公钥：打开 PuttyGen 生成的公钥文件，然后复制即可。

使用 FileZilla 上传文件

安装并打开 FileZilla，打开站点管理，添加新站点，按如下要求填写完成后连接即可。连接之后请将网页文件上传至 /public_html 文件夹内

  • 协议： SFTP
  • 端口： 2222
  • 登入类型： 密钥文件
  • 用户名： jaccount
  • 密钥文件
    • Mac: /Users/USERNAME/.ssh/id_rsa
    • Linux: /home/USERNAME/.ssh/id_rsa
    • Windows: 选择 PuttyGen 生成的私钥文件
